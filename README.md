Project Name :- Sheryan   
Brief description :-  
Sheryan is a smart platform designed to save lives by connecting blood donors, hospitals, and blood banks through a real-time digital network. It aims to eliminate the randomness of searching for blood donors on social media by providing a live map of blood stock levels and an intelligent notification system that alerts nearby donors to urgent cases in real-time.
